# HQL Builder - Latest Changes

## Текущее состояние проекта

### Дата последнего обновления
- **День создания memory-bank**: 2024

### История изменений

#### Версия 1.0.0 (Текущая)

**Функционал:**
- Полная реализация Builder API для HQL запросов
- Поддержка SELECT, UPDATE, DELETE запросов
- Реализация DSL для типобезопасного создания путей свойств
- Поддержка всех типов JOIN (LEFT, RIGHT, INNER, FULL)
- Реализация всех основных выражений (eq, like, between и т.д.)
- Поддержка ORDER BY и GROUP BY
- Поддержка параметризированных запросов

**Изменения API по сравнению с предыдущими версиями:**
- Нет предыдущих версий, это первая стабильная версия

**Известные ограничения:**
- INSERT запросы не поддерживаются из-за ограничений HQL
- Требование аннотации `@Entity` на всех сущностях
- Ограниченная поддержка сложных HQL конструкций
