# HQL Builder - Project Overview

## What is this project?

HQL Builder - это Java-библиотека для упрощения создания HQL (Hibernate Query Language) запросов. Она предоставляет fluent API на основе паттерна Builder и поддерживает domain-specific language (DSL) для безопасного создания запросов без опечаток.

## Основные возможности

- Создание domain-specific представлений сущностей для предотвращения опечаток
- Построение запросов с использованием builder-паттерна и выражений
- Получение готового HQL-запроса и карты параметров
- Поддержка SELECT, UPDATE и DELETE запросов
- Join'ы различных типов (LEFT, RIGHT, INNER, FULL)
- Поддержка DSL и ручного задания путей свойств

## Технологический стек

- Язык: Java
- Сборка: Maven (pom.xml)
- ORM: Hibernate
- Версия: 1.0.0 (находится в разработке/ожидает публикации в Maven Repository)
