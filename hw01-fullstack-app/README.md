# Домашнее задание
Генерация простого приложения в IDE

## Запуск приложения:
- Из корня проекта выполнить ```docker-compose up --build```
- Открыть в браузере ```http://localhost:8080```

## Цель:
Создать минимальное full-stack приложение (backend + frontend) с помощью AI-ассистента в IDE;
настроить API для получения конфигурации анкеты и отправки ответов;
научиться просить ассистента структурировать архитектуру и генерировать отдельные файлы;
развернуть фронтенд, подключенный к backend API;
реализовать хранение пользовательских ответов в памяти backend-приложения;
проверить работу приложения локально.

## Используемые технологии:
vs code, copilot, gpt-4.1, java 21, spring, thymeleaf, docker.

## ПРОМТ 1:

Мы находимся в пустой папке homework_1.
Используемый софт: java 21, spring, thymeleaf.

Необходимо сгенерировать простое приложение «Мини-анкета», состоящее из backend-части и небольшого frontend-интерфейса.

Требуемый функционал:

Backend предоставляет два API:
GET /questions — возвращает список вопросов анкеты (жёстко заданных, 5 шт.).
POST /answers — принимает ответы пользователя и сохраняет их в памяти (в ArrayList).

Frontend:
загружает вопросы с backend;
отображает их в HTML/JS интерфейсе;
отправляет заполненные ответы через POST /answers;
показывает пользователю сообщение «Спасибо!» после отправки.

Техническое уточнение:
Необходимо создать 2 контроллера - Веб и Рест.
Приложение должно открываться по http://localhost:8080
Всё приложение необходимо запустить в одном докер контейнере.
В homework_1/ создать docker-compose файл.
Чтоб я могла из корня проекта сделать docker-compose up, открыть браузер http://localhost:8080 и увидеть анкету.
Так же создай README.md в корне проекта. Там опиши как запустить проект.
